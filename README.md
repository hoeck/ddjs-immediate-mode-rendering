## Dev

To start the slide show:

- `yarn install`
- `yarn run dev`
- visit http://localhost:3030

To build:

- `rm -rf docs`
- `node_modules/.bin/slidev build --base /ddjs-immediate-mode-rendering/`
- `mv dist docs`
- commit and push

Built with [slidev](https://sli.dev/).
